# Introduction

JavaScript (JS) is a lightweight, cross-platform, single-threaded interpreted (or just-in-time compiled) programming language. It is core scripting language in web browsers.

## Key Features

- **Dynamic Typing**: Variables can hold any data type and change during execution
- **Event-Driven**: Perfect for handling user interactions and asynchronous operations
- **Functional Programming**: Supports closures, higher-order functions, and callbacks
- **Object-Oriented**: Prototype-based inheritance and ES6 class syntax
- **Asynchronous Support**: Promises, async/await for non-blocking operations

## Common Use Cases

- Frontend web development with frameworks like React, Vue, and Angular
- Backend development with Node.js
- Mobile app development with React Native and Expo
- Server-side rendering and APIs
- Real-time applications and WebSocket communication

## Execution Environment

JavaScript runs in web browsers (Chrome, Firefox, Safari, Edge) and on servers via Node.js runtime. It has evolved significantly with ECMAScript standards (ES5, ES6/ES2015, and beyond).
